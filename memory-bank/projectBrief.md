# projectBrief.md — Resumen ejecutivo del proyecto

## Nombre del proyecto
**Análisis exploratorio de datos para decisiones comerciales — ComercioYA**

## Institución
Alkemy — Evaluación del módulo 5 (Análisis Exploratorio de Datos)

## Problema a resolver
La empresa ComercioYA (e-commerce) tiene datos históricos de comportamiento de clientes
(compras, visitas, montos, devoluciones, reseñas) pero **no sabe cómo interpretarlos**
para tomar decisiones estratégicas. El equipo directivo necesita claridad sobre qué
variables son relevantes para mejorar ventas y marketing.

## Objetivo principal
Aplicar técnicas de EDA para obtener hallazgos que permitan comprender el comportamiento
de los clientes y sustentar decisiones estratégicas, entregando un informe técnico con
visualizaciones, estadísticas y recomendaciones.

## Alcance del proyecto
6 lecciones progresivas que cubren:
1. EDA inicial e IDA
2. Estadística descriptiva
3. Correlación
4. Regresión lineal
5. Visualización con Seaborn
6. Visualización con Matplotlib

## Entregables finales
- Notebook VSCode comentado y limpio
- Gráficos exportados PNG/PDF
- Informe técnico de análisis (PDF o presentación)
- Código fuente comentado
- Documento de insights y recomendaciones
- Repositorio en GitHub con README

## Criterios de evaluación
- Identificación correcta de tipos de variables
- Aplicación adecuada de medidas estadísticas
- Interpretación correcta de correlaciones y regresiones
- Calidad visual y técnica de los gráficos
- Documentación clara con foco analítico
- Coherencia y profesionalismo del informe final
